# Chat Application

## How to Run:

1. Install Node.js
2. Install dependencies:
    ```
    npm install express socket.io
    ```
3. Run server:
    ```
    node server.js
    ```
4. Open browser at:
    ```
    http://localhost:3000
    ```

## Features:

✅ Join chat rooms  
✅ Real-time messaging  
✅ User authentication (username)  
✅ Message timestamps  
✅ Room management  
✅ Responsive UI
